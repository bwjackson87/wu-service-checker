All content is a lab-safe recreation of real-world problems and techniques encountered in DoD and enterprise environments. Nothing in this repository reflects, discloses, or is derived from any classified, controlled, or otherwise restricted information.

| Item | Status |
|---|---|
| Proprietary employer data | Not included |
| Classified or CUI material | Not included |
| Real hostnames / domain names | Fictionalized (e.g., `corp.local`, `WORKSTATION01`) |
| Real IP addresses | Generalized (e.g., `192.168.10.x` RFC 1918 ranges) |
